# Memory Chain

## Description
The first participant introduces themselves by saying their name together with a word, fact, phrase, or gesture. Each following participant repeats everything that came before in the correct order before adding their own contribution, creating an increasingly longer memory chain.

## Materials Needed
- None

## Time Needed
- 10–15 minutes

## Purpose/Benefit
- Strengthens memory and recall.
- Supports name learning.
- Encourages active listening.
- Improves concentration and attention to detail.
- Builds familiarity and group connection.
