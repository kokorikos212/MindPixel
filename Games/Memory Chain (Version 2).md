# Story Chain

## Description
Participants collaboratively build a story. The first person starts with a short sentence (e.g., *"I woke up."*). Each following participant repeats the entire story from the beginning before adding one new sentence. The story gradually becomes longer, more creative, and often humorous as each participant contributes.

## Materials Needed
- None

## Time Needed
- 10–15 minutes

## Purpose/Benefit
- Develops memory and recall.
- Improves listening skills.
- Encourages creativity and imagination.
- Strengthens teamwork through collaborative storytelling.
- Promotes communication and active participation.
